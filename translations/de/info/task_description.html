Tic-Tac-Toe, manchmal auch als Xs und Os bekannt, ist ein Spiel für zwei Spieler (X und O), die abwechselnd die Felder in einem 3×3-Gitter markieren. Der Spieler, dem es gelingt, drei entsprechende Markierungen in einer horizontalen, vertikalen oder diagonalen Reihe (NW-SE und NE-SW) zu platzieren, gewinnt das Spiel.

Aber wir werden dieses Spiel nicht spielen. Sie werden der Schiedsrichter für die Ergebnisse dieses Spiels sein. Sie erhalten ein Ergebnis eines spiel und Sie müssen entscheiden, ob das Spiel mit einem Sieg oder einem Unentschieden endet und wer der Sieger ist. Stellen Sie sicher, dass Sie zurückgeben "X" wenn der X-Spieler gewinnt und "O", wenn der O-Spieler gewinnt. Wenn das Spiel unentschieden endet, geben Sie "D" zurück.

Eingabe: Ein Spielergebnis als {% if is_js %}Array{% else %}Liste{% endif %} von Zeichenketten (Unicode).
